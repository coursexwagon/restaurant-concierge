# Booking Skill

## Triggers

- "book a table"
- "make a reservation"
- "reserve"
- "I want to come"
- "available"

## Flow

1. Customer wants to book
2. Ask: date, time, number of guests
3. Check availability using tool
4. Confirm booking details
5. Get customer name and phone
6. Confirm booking
7. Thank them

## Examples

Customer: "I want to book a table"
Agent: "I'd love to help you book! What date and time would you like to come, and how many guests?"

Customer: "This Saturday at 7pm for 4 people"
Agent: "Let me check availability for Saturday at 7pm for 4 guests..."

## Important

- Always confirm date/time format
- Get contact details for confirmation
- Use check_availability tool
- Create booking with create_booking tool
